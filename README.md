# 🖼️ 图片抠图转格式工具

一个纯前端的图片处理工具，支持抠图、格式转换、批量导出等功能，无需安装任何软件，打开浏览器即可使用。

![Preview](preview.png)

## ✨ 功能特性

### 🎯 抠图工具
- **点击抠图** — 点击颜色，相似颜色区域自动移除
- **区域抠图** — 点击区域，相连的相似颜色被移除
- **画笔恢复** — 涂抹恢复被抠掉的区域
- **橡皮擦** — 擦除图像区域
- **去白色/黑色背景** — 一键去除纯色背景

### 🔍 缩放与拖动
- **滚轮缩放** — 鼠标指哪放大哪，步进 4%
- **中键拖动** — 按住鼠标中键 360° 拖动画布
- **+/- 按钮** — 精确缩放，步进 10%
- **缩放范围** — 10% ~ 500%

### 💾 多格式导出
- **当前图片** — 导出当前编辑的图片
- **全部图片** — 批量导出所有导入的图片
- **支持格式** — `.png` `.svg` `.webp` `.jpeg` `.ico`

### 📝 批量命名
- **单独命名** — 为单张图片设置自定义名称
- **批量命名** — 一键为所有图片设置基础名称，自动编号：名称(1)、名称(2)…

### 📐 尺寸调整
- 预设尺寸：16×16、32×32、64×64、128×128、256×256
- 图片默认尺寸（原始大小）
- 自定义尺寸

### ⌨️ 快捷键
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` | 撤销 |
| 鼠标中键拖动 | 拖动画布 |
| 滚轮 | 缩放 |
| 右键 | 复制图片 |
| `Ctrl+V` | 粘贴图片导入 |

## 🚀 快速开始

### 在线使用
直接打开 `image-to-ico.html` 即可使用，无需服务器。

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/your-username/image-to-ico.git

# 打开 HTML 文件
open image-to-ico.html
```

## 📖 使用说明

1. **导入图片** — 拖拽图片到页面任意位置，或点击「📂 打开文件」按钮
2. **抠图** — 选择工具（点击抠图/区域抠图），点击图片上的背景颜色
3. **调整** — 使用画笔恢复/橡皮擦微调边缘
4. **缩放** — 滚轮放大缩小查看细节，中键拖动查看不同区域
5. **导出** — 选择导出模式（当前/全部/批量命名）和格式，点击「📥 导出」

## 🖥️ 技术栈

- 纯 HTML + CSS + JavaScript，无任何依赖
- Canvas API 实现图像处理
- CSS Grid + Flexbox 响应式布局
- 暗色主题设计

| 导出 | PNG、SVG、WebP、JPEG、ICO |
