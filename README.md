# 🖼️ 图片转ICO工具

一个纯前端的图片转ICO工具，支持抠图、背景移除、尺寸调整和ICO格式导出。

## ✨ 功能特性

- 🎯 **点击抠图** - 点击图片上的颜色，自动移除相似颜色区域
- 🌊 **区域抠图** - 智能识别相连区域并移除
- 🖌️ **画笔恢复** - 可恢复被误删的图像区域
- 🧹 **橡皮擦** - 手动擦除不需要的部分
- ⚡ **一键去背景** - 快速去除白色/黑色背景
- 📐 **多尺寸支持** - 16×16 到 256×256，支持自定义尺寸
- 💾 **多格式导出** - 支持导出 PNG 和 ICO 格式

## 🚀 使用方法

1. 直接在浏览器中打开 `index.html`
2. 拖拽或选择图片上传
3. 使用工具栏进行抠图编辑
4. 选择目标尺寸
5. 导出为 PNG 或 ICO 格式

## 📝 使用提示

- **点击抠图**：点击图片上的背景颜色，容差内的相似颜色会被移除
- **区域抠图**：点击一个区域，相连的相似颜色会被移除
- **画笔恢复**：按住鼠标左键涂抹，可恢复原图内容
- **橡皮擦**：按住鼠标左键涂抹，可擦除图像区域
- **边缘平滑**：勾选后可让抠图边缘更自然
